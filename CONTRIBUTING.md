# Contributing to Merq

Thanks for your interest in Merq 🎉

## Ways to contribute
- Bug reports
- Feature requests
- Documentation improvements
- Code contributions

## Development setup

1. Fork the repo
2. Install dependencies
3. Run migrations
4. Start the dev server

```bash
npm install
npm run migrate
npm run dev
```
## Code style
- TypeScript only
- No any
- Prefer pure functions
- Small files > large files

## Pull requests
- One feature per PR
- Add tests where possible
- Describe why, not just what
## **Code of Conduct**
Be respectful. No harassment.
